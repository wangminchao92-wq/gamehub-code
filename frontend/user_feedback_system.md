# GameHub 用户反馈收集系统

## 系统概述
为 GameHub 用户测试设计的完整反馈收集、分析和处理系统。

## 反馈收集渠道

### 1. 在线问卷系统
**工具**: 问卷星 / Google Forms 替代方案
**频率**: 每个测试场景结束后
**内容**: 定量评分 + 定性反馈

### 2. 实时反馈按钮
**位置**: 网站右下角固定按钮
**功能**: 一键反馈当前页面问题
**数据**: 页面URL、用户角色、时间戳、反馈内容

### 3. 测试观察记录
**方法**: 屏幕录制 + 观察员记录
**工具**: OBS Studio + 记录表格
**内容**: 用户行为、表情、口头反馈

### 4. 深度访谈
**对象**: 代表性用户 (3-5人)
**时间**: 测试结束后30分钟
**内容**: 深入体验、改进建议、长期期望

## 反馈收集流程

### 阶段一：收集 (测试期间)
```
用户测试 → 填写问卷 → 实时反馈 → 观察记录 → 深度访谈
      ↓          ↓          ↓          ↓          ↓
  数据收集 → 数据整理 → 初步分类 → 优先级评估
```

### 阶段二：分析 (测试后1天)
```
原始数据 → 定量分析 → 定性分析 → 问题分类 → 趋势识别
    ↓          ↓          ↓          ↓          ↓
数据清洗 → 统计计算 → 文本分析 → 模式识别 → 洞察提炼
```

### 阶段三：处理 (测试后2天)
```
问题清单 → 优先级排序 → 分配处理 → 解决方案 → 验证测试
    ↓          ↓          ↓          ↓          ↓
严重问题 → 立即修复 → 用户体验 → 功能优化 → 回归测试
```

## 数据分析指标

### 定量指标
1. **整体满意度** (CSAT)
   - 计算公式: (4-5分数量 / 总数量) × 100%
   - 目标值: ≥ 80%

2. **净推荐值** (NPS)
   - 计算公式: (推荐者% - 贬损者%) × 100
   - 目标值: ≥ 50

3. **任务完成率**
   - 计算公式: (完成任务用户 / 总用户) × 100%
   - 目标值: ≥ 90%

4. **平均任务时间**
   - 计算公式: 总任务时间 / 完成任务数
   - 基准值: 各场景设定合理时间

### 定性指标
1. **问题频率分析**
   - 高频问题: 出现次数 > 总用户数 × 30%
   - 中频问题: 出现次数 > 总用户数 × 10%
   - 低频问题: 出现次数 ≤ 总用户数 × 10%

2. **情感分析**
   - 正面情感: 喜欢、满意、惊喜
   - 中性情感: 一般、可接受
   - 负面情感: 困惑、沮丧、失望

3. **建议分类**
   - 功能建议: 新增或改进功能
   - 体验建议: 优化交互流程
   - 界面建议: 调整视觉设计
   - 性能建议: 提升加载速度

## 反馈处理优先级

### P0: 紧急问题 (24小时内处理)
- 严重bug导致功能无法使用
- 安全漏洞或数据丢失风险
- 核心功能完全失效

### P1: 高优先级 (3天内处理)
- 影响主要用户流程的问题
- 多数用户报告的困惑点
- 性能问题影响用户体验

### P2: 中优先级 (1周内处理)
- 界面布局或视觉问题
- 次要功能优化建议
- 用户体验细节改进

### P3: 低优先级 (2周内或后续版本)
- 锦上添花的功能建议
- 非核心体验优化
- 未来版本规划功能

### 3. 数据收集脚本
```javascript
// 数据收集脚本
async function collectFeedback(data) {
  const payload = {
    ...data,
    timestamp: new Date().toISOString(),
    userAgent: navigator.userAgent,
    screenSize: `${window.innerWidth}x${window.innerHeight}`,
    pageUrl: window.location.href,
  };
  
  try {
    const response = await fetch('/api/feedback', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(payload),
    });
    
    return await response.json();
  } catch (error) {
    console.error('反馈提交失败:', error);
    // 降级方案：存储到localStorage
    localStorage.setItem('pending_feedback', JSON.stringify(payload));
  }
}
```
