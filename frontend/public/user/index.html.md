# GameHub用户中心

## 用户中心概述

GameHub用户中心是每位玩家的个人空间，提供个人资料管理、成就展示、活动记录、社交互动等全方位功能。我们致力于为每位玩家打造个性化、有价值的个人体验。

## 个人资料

### 基本信息
- **用户名**: 唯一的社区标识
- **头像系统**: 自定义头像和头像框
- **个人简介**: 简短的个人介绍和签名
- **社交链接**: 社交媒体和个人网站链接

### 详细资料
- **游戏偏好**: 喜欢的游戏类型和平台
- **游戏经历**: 游戏历史和成就
- **地理位置**: 可选的地理位置信息
- **联系方式**: 可选的联系方式（隐私保护）

### 隐私设置
- **资料可见性**: 控制个人资料的可见范围
- **活动隐私**: 控制活动记录的隐私设置
- **消息权限**: 控制谁可以发送消息
- **数据导出**: 个人数据导出和删除
