# GameHub 新闻模块

## 模块概述

新闻模块是GameHub的核心功能之一，提供最新的游戏行业新闻、游戏发布信息、更新动态和行业趋势分析。

## 功能特点

### 1. 新闻分类
- **行业动态**: 游戏行业新闻和趋势
- **游戏发布**: 新游戏发布和预告
- **更新信息**: 游戏更新和补丁说明
- **活动报道**: 游戏展会和活动报道
- **开发者访谈**: 游戏开发者采访和幕后故事

### 2. 内容管理
- **每日更新**: 保持内容新鲜度
- **多来源**: 官方公告、媒体报道、社区消息
- **编辑审核**: 专业编辑内容审核
- **时间线**: 按时间顺序展示新闻

### 3. 用户交互
- **阅读统计**: 文章阅读次数统计
- **点赞收藏**: 用户点赞和收藏功能
- **评论系统**: 用户评论和讨论
- **分享功能**: 社交媒体分享

## 技术架构细节

### 前端组件
```typescript
// 新闻列表组件
interface NewsItem {
  id: string;
  title: string;
  summary: string;
  category: string;
  publishDate: string;
  readCount: number;
  likeCount: number;
  commentCount: number;
  imageUrl: string;
}

// 新闻详情组件
interface NewsDetail extends NewsItem {
  content: string;
  author: string;
  tags: string[];
  relatedNews: NewsItem[];
}
```

### API接口
```
GET /api/news           # 获取新闻列表
GET /api/news/{id}      # 获取新闻详情
GET /api/news/categories # 获取分类列表
POST /api/news/{id}/like # 点赞新闻
POST /api/news/{id}/comment # 发表评论
```

### 数据库设计
```sql
-- 新闻表
CREATE TABLE news (
  id UUID PRIMARY KEY,
  title VARCHAR(255) NOT NULL,
  content TEXT NOT NULL,
  category_id UUID REFERENCES categories(id),
  author_id UUID REFERENCES users(id),
  publish_date TIMESTAMP,
  status VARCHAR(20) DEFAULT 'draft'
);

-- 分类表
CREATE TABLE categories (
  id UUID PRIMARY KEY,
  name VARCHAR(50) NOT NULL,
  slug VARCHAR(50) UNIQUE NOT NULL,
  description TEXT
);
```
