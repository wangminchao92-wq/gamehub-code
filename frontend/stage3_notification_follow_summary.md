
## 🚀 **立即可用的功能**

### 1. **通知系统**
```typescript
// 使用示例
<NotificationSystem
  userId="user123"
  onMarkAsRead={handleMarkAsRead}
  onMarkAllAsRead={handleMarkAllAsRead}
  onDelete={handleDelete}
  realTimeEnabled={true}
  pollingInterval={30000}
  showUnreadOnly={false}
  requireAuth={true}
/>
```

### 2. **通知API**
```bash
# 获取用户通知
GET /api/notifications

# 获取未读通知
GET /api/notifications?unreadOnly=true

# 标记通知为已读
POST /api/notifications
{
  "action": "mark-read",
  "notificationIds": ["notif-001", "notif-002"]
}

# 获取通知设置
POST /api/notifications
{
  "action": "get-settings"
}
```

### 3. **关注系统**
```typescript
// 使用示例
<FollowSystem
  userId="user123"
  mode="FOLLOWERS"
  onFollow={handleFollow}
  onUnfollow={handleUnfollow}
  onBlock={handleBlock}
  showStats={true}
  showActions={true}
  showSearch={true}
  requireAuth={true}
/>
```

### 4. **关注API**
```bash
# 获取关注者
GET /api/follow?mode=FOLLOWERS

# 获取正在关注的用户
GET /api/follow?mode=FOLLOWING

# 获取互相关注的用户
GET /api/follow?mode=MUTUAL

# 获取推荐用户
GET /api/follow?mode=SUGGESTIONS

# 关注用户
POST /api/follow
{
  "action": "follow",
  "targetUserId": "user456"
}
```
